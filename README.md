## Hirefy Challenge

### El challenge consiste en replicar una barra de navegación vertical
#### Librerías y herramientas usadas:
- React
- React Router
- Styled Components
- Material UI

#### Instalacion:
- Clonar proyecto
- Instala las dependencias: ```npm install```
- Corre el ambiente local: ```npm run dev```
